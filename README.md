cucumber-salad
==============

## Quick start

1. Install nvm with the command: 

   `curl https://raw.githubusercontent.com/creationix/nvm/v0.18.0/install.sh | bash`
   
2. Install dependencies

  `npm install`

3. Run the tests 

  `grunt cucumberjs`

4. Check code style

  `grunt jshint`
